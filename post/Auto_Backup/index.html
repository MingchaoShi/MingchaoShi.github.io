电脑文件的本地及云端自动备份

2019-02-21

这是我一直在用的自动备份方案，一经设置可自动实现指定目录的本地及云端备份，非常方便。

通过下面的设置，PC将在指定的时间，将指定目录(A处)的内容自动同步到本地另一处(B处)，并且同步到云盘，实现无人值守的自动备份功能。 注1：以下环境为Windows7，网络云盘为坚果云。 注2：设需要备份的文件位于目录A，本地备份于目录B，目录B同时也是坚果云的同步目录。 注3：坚果云的同步应始终处于运行状态。

下载安装本地同步工具SyncToy，这是微软出品的免费工具；（下载链接：https://www.microsoft.com/en-us/download/details.aspx?id=15155） 配置SyncToy： Create New Folder Pair：在Left Folder里面输入目录A的地址，在Right Folder里面输入目录B的地址； 在接下来的页面，选择“Echo”; 在接下来的页面，设置此同步的名字，比如“Documents Backup”。 设置任务计划程序：附件→系统工具→任务计划程序新建一个任务计划，在指定的时间运行SyncToy程序，比如："C:\Program Files\SyncToy 2.1\SyncToyCmd.exe"，并添加参数 -R。你可以自由设定备份时间和频率。 注册并下载安装坚果云，将目录B设置为坚果云的同步目录。
